# Cost Finance AI Agent

Enterprise-style AI Agent for cost estimation, budget tracking,
and finance analytics in construction/project systems.

## Current Version (Milestone 1)

### Features implemented:
- FastAPI agent endpoint (/agent)
- LangChain-based finance agent
- Tool routing (subsystem_cost, cost_breakdown)
- Mock finance dataset integration
- Working test harness (test_agent.py)
- JSON-based LLM decision parsing
- Basic error handling
